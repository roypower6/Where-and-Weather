# **Where and Weather**

> **💡 OpenWeatherMap API를 이용한 날씨 조회 어플리케이션**

## **📱 소개**
OpenWeatherMap 무료 플랜 API를 이용한 현재 위치 기반, 검색 날씨 조회 어플리케이션입니다.


## **✨ 주요 기능**
